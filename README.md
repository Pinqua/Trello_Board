# Trello Board

## Demo

https://trello-board.vercel.app


## Run Locally

Clone the project

```bash
  git clone https://github.com/Pinqua/Trello_Board.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
