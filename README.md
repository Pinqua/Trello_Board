# Trello Board

## Demo

https://trello-board-pinqua.vercel.app


## Run Locally

Clone the project

```bash
  git clone https://github.com/Pinqua/Trello_Board.git
```

Go to the project directory

```bash
  cd Trello_Board
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
